Tóm tắt Internet và Web

Internet

Internet là mạng lưới kết nối các máy tính trên toàn thế giới, giúp người dùng trao đổi và truy cập thông tin.

Web

Web là dịch vụ hoạt động trên Internet, cho phép người dùng truy cập thông tin thông qua trình duyệt web.

Website

Website là tập hợp nhiều trang web có liên quan với nhau và được lưu trữ trên máy chủ web.

Trang web

Trang web là một trang thông tin đơn lẻ trên website, thường được viết bằng HTML.
